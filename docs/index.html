Everything you need. Nothing you don't.

🔍

Auto-detect

Scans project files — package.json, Gemfile, go.mod, composer.json — and resolves the full dependency graph automatically.

⚡

Native Install

Installs services directly on your OS via Homebrew, apt, or winget. No container layers. Native performance, always.

🛡️

Isolation Cells

OS-native sandboxing — macOS Sandbox, Linux namespaces, Windows AppContainers. Per-project isolation without VMs.

🤖

AI Assistant

Built-in AI that reads your logs, suggests fixes, and explains errors. Cascading model fallback: local → API → cloud.

🚀

Deploy

Generate Terraform configs and Ansible playbooks from your rawenv.toml. One command to go from dev to production.

🌐

Tunneling

Expose local services with public URLs. Built-in reverse proxy, custom domains, and automatic TLS. Share instantly.
